Modelos de Servicio de cómputo en la nube

Infraestructura como Servicio (IaaS): En este modelo, se proporciona una infraestructura de cómputo virtualizada, que incluye servidores, redes, almacenamiento y otros recursos. Los usuarios pueden aprovisionar y administrar estos recursos según sea necesario. Ejemplos de proveedores de IaaS incluyen Amazon Web Services (AWS), Microsoft Azure y Google Cloud Platform. Plataforma como Servicio (PaaS): En este modelo, se proporciona una plataforma de desarrollo y ejecución de aplicaciones en la nube. Los usuarios pueden desarrollar, implementar y administrar aplicaciones sin preocuparse por la infraestructura subyacente. Ejemplos de PaaS incluyen Microsoft Azure App Service y Google App Engine. Software como Servicio (SaaS): En este modelo, se ofrece software completamente funcional a través de la nube. Los usuarios acceden a la aplicación a través de un navegador web y no necesitan preocuparse por la administración de la infraestructura ni las actualizaciones del software. Ejemplos de SaaS son servicios como Microsoft 365, Salesforce y Dropbox.
